# Pokémon MongoDB Project

## Overview
This project builds a data pipeline using Python and MongoDB to fetch, transform, and analyze Pokémon data from the PokeAPI.

## Goals
- Practice API data extraction
- Store and model data in MongoDB
- Build aggregation pipelines
- Structure a clean data project

## Project Structure
- `notebook/` → Jupyter Notebook for analysis
- `src/` → Python scripts (API, transform, load)
- `data/` → raw/sample data
- `images/` → diagrams (class diagram, data flow)
